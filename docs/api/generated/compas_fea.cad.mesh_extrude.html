mesh_extrude

compas_fea.cad.mesh_extrude(structure, guid, layers, thickness, mesh_name='', links_name='', blocks_name='', points_name='', plot_mesh=False, plot_links=False, plot_blocks=False, plot_points=False)

Extrudes a Rhino mesh and adds/creates elements.

Parameters

  • structure (obj) – Structure object to update.

  • guid (guid) – Rhino mesh guid.

  • layers (int) – Number of layers.

  • thickness (float) – Layer thickness.

  • mesh_name (str) – Name of set for mesh on final surface.

  • links_name (str) – Name of set for adding links along extrusion.

  • blocks_name (str) – Name of set for solid elements.

  • points_name (str) – Name of aded points.

  • plot_mesh (bool) – Plot outer mesh.

  • plot_links (bool) – Plot links.

  • plot_blocks (bool) – Plot blocks.

  • plot_points (bool) – Plot end points.

Returns

None

Notes

  • Extrusion is along the mesh vertex normals.
