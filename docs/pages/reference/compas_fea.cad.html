compas_fea.cad ¶

The compas_fea package supports Rhino and Blender.

blender ¶

compas_fea.cad.blender

add_nodes_elements_from_bmesh (structure, bmesh) Adds the Blender mesh’s nodes, edges and faces to the Structure object.
add_nodes_elements_from_layers (structure, layers) Adds node and element data from Blender layers to Structure object.
add_tets_from_bmesh (structure, name, bmesh) Adds tetrahedron elements from a Blender mesh to Structure object.
add_nset_from_bmeshes (structure, name[, ...]) Adds the Blender meshes’ vertices as a node set.
add_elset_from_bmeshes (structure, name[, ...]) Adds the Blender meshes’ edges and faces as an element set.
add_nset_from_objects (structure, name[, ...]) Adds the objects’ locations as a node set.
plot_data (structure, path, name, step[, ...]) Plots analysis results on the deformed shape of the Structure.
ordered_lines (structure, network, layer) Extract node and element orders from a Network for a given start-point.
plot_voxels (structure, path, name, step[, ...]) Applies a base voxel material and texture to a cube for 4D visualisation.
mesh_extrude (structure, bmesh, nz, dz) Extrudes a Blender mesh into cells of many layers.

rhino ¶

compas_fea.cad.rhino

add_element_set (structure, guids, name[, ...]) Adds element set information from Rhino curve and mesh guids.
add_node_set (structure, guids, name[, explode]) Adds node set information from Rhino point guids.
add_nodes_elements_from_layers (structure, ...) Adds node and element data from Rhino layers to Structure object.
add_sets_from_layers (structure, layers[, ...]) Add node or element sets to the Structure object from layers.
mesh_extrude (structure, guid, nz, dz) Extrudes a Rhino mesh into cells of many layers.
network_from_lines (guids) Creates a Network datastructure object from a list of curve guids.
ordered_lines (structure, network, layer) Extract node and element orders from a Network for a given start-point.
plot_axes (xyz, e11, e22, e33[, sc]) Plots a set of axes.
plot_data (structure, path, name, step[, ...]) Plots analysis results on the deformed shape of the Structure.
plot_principal_stresses (structure, path, ...) Plots the principal stresses of the elements.
