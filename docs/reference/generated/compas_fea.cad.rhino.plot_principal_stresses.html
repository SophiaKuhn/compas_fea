plot_principal_stresses

compas_fea.cad.rhino.plot_principal_stresses(structure, step, ptype, scale, rotate=0, layer=None)[source]

Plots the principal stresses of the elements.

Parameters

  • structure (obj) – Structure object.

  • step (str) – Name of the Step.

  • ptype (str) – ‘max’ or ‘min’ for maximum or minimum principal stresses.

  • scale (float) – Scale on the length of the line markers.

  • rotate (int) – Rotate lines by 90 deg, 0 or 1.

  • layer (str) – Layer name for plotting.

Returns

None

Notes

  • Currently an alpha script and only for triangular shell elements in Abaqus.

  • Centroids are taken on the undeformed geometry.
