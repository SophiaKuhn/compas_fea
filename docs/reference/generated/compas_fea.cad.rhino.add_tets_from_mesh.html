add_tets_from_mesh

compas_fea.cad.rhino.add_tets_from_mesh(structure, name, mesh, draw_tets=False, volume=None, layer='Default', thermal=False)[source]

Adds tetrahedron elements from a mesh in Rhino to the Structure object.

Parameters

  • structure (obj) – Structure object to update.

  • name (str) – Name for the element set of tetrahedrons.

  • mesh (guid) – The mesh in Rhino representing the outer surface.

  • draw_tets (str) – Layer to draw tetrahedrons on.

  • volume (float) – Maximum volume for each tet.

  • thermal (bool) – Thermal properties on or off.

Returns

None
