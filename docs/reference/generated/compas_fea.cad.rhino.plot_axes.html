plot_axes

compas_fea.cad.rhino.plot_axes(xyz, e11, e22, e33, layer, sc=1)[source]

Plots a set of axes.

Parameters

  • xyz (list) – Origin of the axes.

  • e11 (list) – Normalised first axis component [x1, y1, z1].

  • e22 (list) – Normalised second axis component [x2, y2, z2].

  • e33 (list) – Normalised third axis component [x3, y3, z3].

  • layer (str) – Layer to plot on.

  • sc (float) – Size of the axis lines.

Returns

None
