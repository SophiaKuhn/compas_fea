cad

The compas_fea package supports Rhino and Blender.

blender

compas_fea.cad.blender

add_nodes_elements_from_bmesh(structure, bmesh) Adds the Blender mesh’s nodes, edges and faces to the Structure object.
add_nodes_elements_from_layers(structure, layers) Adds node and element data from Blender layers to Structure object.
add_tets_from_bmesh(structure, name, bmesh) Adds tetrahedron elements from a Blender mesh to the Structure object.
add_nset_from_bmeshes(structure, name[, ...]) Adds the Blender meshes’ vertices as a node set.
add_elset_from_bmeshes(structure, name[, ...]) Adds the Blender meshes’ edges and faces as an element set.
add_nset_from_objects(structure, name[, ...]) Adds the objects’ locations as a node set.
plot_data(structure, step[, field, layer, ...]) Plots analysis results on the deformed shape of the Structure.
ordered_network(structure, network, layer) Extract node and element orders from a Network for a given start-point.
plot_voxels(structure, step[, field, layer, ...]) Applies a base voxel material and texture to a cube for 4D visualisation.
mesh_extrude(structure, bmesh, nz, dz, setname) Extrudes a Blender mesh into cells of many layers and adds to Structure.
