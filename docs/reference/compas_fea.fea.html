fea

The compas_fea package supports ABAQUS, ANSYS and OpenSEES as analysis backends.

abaq

compas_fea.fea.abaq

abaqus_launch_process(structure, exe, cpus) Runs the analysis through Abaqus.
extract_odb_data(structure, fields, exe) Extract data from the Abaqus .odb file.
input_write_constraints(f, constraints) Writes the constraints information to the Abaqus .inp file.
input_write_elements(f, elements) Writes the element information to the Abaqus .inp file.
input_generate(structure, fields[, units]) Creates the Abaqus .inp file from the Structure object.
input_write_heading(f) Creates the Abaqus .inp file heading.
input_write_materials(f, materials) Writes materials to the Abaqus .inp file.
input_write_misc(f, misc) Writes misc class info to the Abaqus .inp file.
input_write_nodes(f, nodes, units) Writes the nodal co-ordinates information to the Abaqus .inp file.
input_write_properties(f, sections, ...) Writes the section information to the Abaqus .inp file.
input_write_sets(f, sets) Creates the Abaqus .inp file node sets NSETs and element sets ELSETs.
input_write_steps(f, structure, steps, ...) Writes step information to the Abaqus .inp file.
