compas_fea.fea

The compas_fea package supports ABAQUS, ANSYS and OpenSEES as analysis backends.

abaq

compas_fea.fea.abaq

inp_write_constraints(f, constraints) Writes the constraints information to the Abaqus .inp file.
inp_write_elements(f, elements) Writes the element information to the Abaqus .inp file.
inp_generate(structure, filename[, units]) Creates the Abaqus .inp file from the Structure object.
inp_write_heading(f) Creates the Abaqus .inp file heading.
inp_write_materials(f, materials) Writes materials to the Abaqus .inp file.
inp_write_misc(f, misc) Writes misc class info to the Abaqus .inp file.
inp_write_nodes(f, nodes, units) Writes the nodal co-ordinates information to the Abaqus .inp file.
inp_write_properties(f, sections, ...) Writes the section information to the Abaqus .inp file.
inp_write_sets(f, sets) Creates the Abaqus .inp file node sets NSETs and element sets ELSETs.
inp_write_steps(f, steps, loads, ...) Writes step information to the Abaqus .inp file.
